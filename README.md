# TestCreator_React

Test creator web app (ASP.NET Core 3.1 and React)

Used technologies and tools:

Back-end
- ASP.NET Core 3.1
- EntityFramework Core 3.1
- NUnit
- AutoMapper
- Autofac
- FluentAssertions
- Moq
- AutoFixture

Front-end
- react 16.11
- redux 7.1
- redux-form
- axios
- MaterialUI 
- Fortawesome Icons
